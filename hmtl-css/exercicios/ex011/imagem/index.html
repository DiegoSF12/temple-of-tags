Testando o dinamismo das imagens

para que possamos adequar o tamanho da imagem de forma dinamica e automatica podemos usar a tag <picture>; e dentro dele devemos usar a tag <source>

a tag <source>; deve ter alguns atributos junto a ela como:


media=""
Aqui deve ser especificado o tamanho maximo que o dispositivo deve ter, como por exeplo media="(max=width:350px)"

srcset=""
Aqui devemos especificar o arquivo que será carregado caso a condição media seja atendida

type=""
Aqui devemos definir o tipo de arquivo que será carregado, você pode conferir os tipos de arquivos em:https://www.iana.org/assignments/media-types/media-types.xhtml
