Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usado a tag <b>;(não semantica)

Nesta frase temos um termo em destaque com a tag <strong>(semantica)

Itálico / Ênfase

Nesta frase, temos um termo em italico usando a tag <i> (não semantica)

Nesta frase temos um termo em ênfase usando a tag <em> (semantica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Temos um outro texto que foi marcado com a mesma tag

Texto grande e texto pequeno

Estamos criando um texto grande com a tag <big>(tag obsoleta) e um texto pequeno com a tag <small>

Texto deletado

Podemos marcar um texto como exluido para indicar que ele deve ser lido, mas não considerado, usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adiconado depois, com a tag <ins>

Existe também o sublinhado com a tag <u>(não semantica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag <sup>

Texto Subescrito

Para inserir coisas do tipo H2O usamos a tag <sub>
